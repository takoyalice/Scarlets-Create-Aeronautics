# Scarlet's Create Aeronautics Modpack Installation Guide

## Use the Modrinth App

I recommend using the **Modrinth App** for installation. It works similarly to CurseForge but makes setting up custom servers much easier.

[**Download the Modrinth App**](https://modrinth.com/app)

> Modrinth App is an open source launcher that allows you to play your favorite mods and keep them up to date, all in one neat little package. Very safe.

You can also use the [**Prism Launcher**](https://prismlauncher.org/), which can handle both CurseForge modpacks and Modrinth .mrpacks

## Installation

1. Head to [Releases](https://github.com/takoyalice/Scarlets-Create-Aeronautics/releases/latest) and download the latest .mrpack binary file. (essentially a ZIP file of the entire modpack).
2. Double-click the file to open it within the Modrinth App. 
3. Install the profile.
4. Done !

## Memory Settings

After installing the modpack, press the **Gear** icon to modify the amount of dedicated memory for the modpack.

> Not sure what value is best for your setup - adjust as needed. I use around 10 Gigabytes.
